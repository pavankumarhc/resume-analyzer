# 📄 Resume Analyzer

## 📌 Description
A web application built using Python and Streamlit that analyzes resumes by extracting text, detecting skills, and providing a resume score.

## 🚀 Features
- Upload PDF Resume
- Extract Resume Content
- Word Count
- Skills Detection
- Resume Score
- Suggestions for Improvement
- Job Role Recommendation

## 🛠️ Technologies Used
- Python
- Streamlit
- PyPDF2

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py


## 🎯 Output
- Extracted resume text
- Skills detected
- Resume score
