📄 AI Resume Analyzer

An intelligent Resume Analyzer built with Python and Streamlit that helps users evaluate resumes instantly. Upload a PDF resume and get smart insights like resume score, skills detection, keyword analysis, charts, and improvement suggestions.

---

🚀 Features

✅ Upload Resume in PDF format
✅ Extract text from PDF automatically
✅ Detect technical skills from resume
✅ Resume Score out of 100
✅ Email & Phone Number Detection
✅ Smart Suggestions for improvement
✅ Skills Comparison Graph
✅ Resume Composition Pie Chart
✅ Top Keywords Bar Chart
✅ Clean and Professional UI

---

🛠️ Technologies Used

- Python
- Streamlit
- PyPDF2
- Matplotlib
- Regular Expressions (re)
- Collections (Counter)
